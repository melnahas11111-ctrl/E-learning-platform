# 🎓 ACES Training Platform — ISO 2859-1

منصة تدريبية متكاملة لمعايير سحب العينات ISO 2859-1:1999  
وزارة الصحة — دولة قطر

## المميزات

- ✅ **لوحة تحكم إدارية** مع إحصائيات فورية
- 📱 **QR Code** لكل متدرب للدخول الفوري
- 🔴 **مراقبة مباشرة** للمتدربين أثناء الاختبار
- 💾 **حفظ تلقائي** — المتدرب يكمل من حيث توقف
- 📊 **نتائج فورية** تظهر للمدرب عند الانتهاء
- 🌐 **ثنائي اللغة** عربي / إنجليزي

## التقنيات

- **Frontend**: HTML + Tailwind CSS
- **Database**: Firebase Firestore (Real-time)
- **Hosting**: GitHub Pages
- **QR Codes**: qrcode.js
